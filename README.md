## About Me

I enjoy building systems that sit underneath products rather than products themselves.

My interests span developer infrastructure, security engineering, AI-assisted software development, backend architecture, observability, monitoring systems, and application reliability.

Over the last few years I've worked across startup environments, security reviews, developer tooling, technical content, integrations, and full-stack product development.

Most recently I've been building **Nightlamp**, an application maintenance platform that helps teams understand dependency updates, security advisories, changelog impacts, and runtime incidents before they become production problems.
